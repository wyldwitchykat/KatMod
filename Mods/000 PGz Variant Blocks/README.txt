PGz modlets are created and written by Phys1csGamez.

"000 PGz Variant Blocks" is a modlet that implements the PGz variant block helper menu.
This modlet must be installed in order to use any PGz block/shapes modlets.
Each block/shape modlet pack includes this modlet. Only one copy is required.
The naming of this modlet is important in order to have it load before the main block/shape modlets.

Videos to support the installation of PGz modlets can be found at https://www.youtube.com/@Phys1csGamez
