-- Fancy Home Deco --7 days to die v 1.0 mod - by LittleRedSonja --

Installation Instructions:

Extract the downloaded zip into your 7D2D Mods folder (create one if it does not exist).

To verify you have installed the mod correctly, the ModInfo.xml file should be located at Mods\LittleRedSonja_FancyHomeDeco_v1\ModInfo.xml relative to your 7D2D install folder.

This mod is Server-Only-Friendly (for multiplayer the mod nees to be installed only in the server, player do not need to install it).
